# Markdown Report Generator

Synthesize multiple sources into professional Markdown reports.

## When to Use

- Synthesizing research findings
- Creating analysis reports
- Compiling multi-source summaries
- Generating structured documentation

## Report Structure

```markdown
# [Report Title]

> **Date:** YYYY-MM-DD
> **Prepared by:** [Author]
> **Classification:** [Internal/Public/Confidential]

## Executive Summary

[2-3 paragraph overview of key findings and recommendations]

## Table of Contents

[Auto-generated]

## 1. Introduction

[Context and scope]

## 2. Methodology

[How the analysis was conducted]

## 3. Findings

### 3.1 [Finding Category 1]
[Details with supporting data]

### 3.2 [Finding Category 2]
[Details with supporting data]

## 4. Analysis

[Interpretation of findings]

## 5. Recommendations

[Actionable next steps]

## 6. Appendices

[Supporting data, charts, references]

## References

[All sources cited]
```

## Formatting Rules

- Use `#` for title, `##` for sections, `###` for subsections
- Tables for structured comparisons
- Code blocks for technical content
- Blockquotes for key insights
- Horizontal rules (`---`) between major sections
- Consistent date formatting (YYYY-MM-DD)

## Quality Standards

- Every claim backed by a source
- Data presented with context
- Charts referenced in text
- Executive summary standalone
- Recommendations specific and actionable
